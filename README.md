# Shiraz Ali | Portfolio

Welcome to the source code of my personal portfolio — a high-performance, visually rich showcase of my work as a Frontend Developer and Agentic AI Engineer.

## Tech Stack
- **Framework:** Next.js 16 (App Router), React 19, TypeScript
- **Styling:** Tailwind CSS 4
- **Animations:** GSAP (with `@gsap/react`), ScrollTrigger, Lenis
- **3D Graphics:** Three.js, `@react-three/fiber`
- **Content:** Sanity CMS

## Key Features
- **Cinematic Frontend:** 3D web experiences and smooth scrolling.
- **Autonomous Blog:** Fully automated weekly blog system that researches, writes, and publishes content using Gemini and Sanity.
- **Agentic-First:** Built with custom AI skills and agentic workflows.

## Getting Started
1. `npm install`
2. Configure required environment variables (see `GEMINI.md`).
3. `npm run dev`
